# Servo_controler_PWM

Project of 6 channel PWM generator on Atmega328p-pu with 2x16 lcd display

Value setting:
- manual (2 buttons) 
- I2C communication

Optionality:
- selection of ports 1-6 for PWM signal generation
- value set in degrees 0°-180° of servo rotation
